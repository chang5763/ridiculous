# 赤壁之戰 · 3D 戰史運鏡特輯

以電視特別節目的 **3D 運鏡與特效**重現西元 208 年(建安十三年)的**赤壁之戰**。單一 HTML 檔、純前端(Three.js),可互動探索,也可自動播放。

🔗 **線上觀看:** https://chang5763.github.io/ridiculous/

## 特色

- **三方陣營辨識**：曹操軍（藍）、孫權軍（紅）、劉備軍（綠），軍旗、船帆、戰力卡顏色一致。
- **3D 地形與地名**：長江、赤壁紅崖、烏林水寨、江陵、夏口、樊口、華容道、七星壇皆有標示。
- **十幕時間軸**：曹操南征 → 孫劉聯盟 → 烏林對峙 → 反間計 → 草船借箭 → 連環計 → 苦肉計詐降 → 借東風 → 火燒赤壁 → 敗走華容。
- **計策發動特效**：反間、連環、苦肉、借東風、火攻皆有大字幕與場景效果。
- **動態特效**：水波、飄揚軍旗、箭雨齊射、濃霧、東南風流場、火船突擊、烈焰濃煙與火箭火砲。
- **各軍狀態**：兵力長條、船艦數、將領姓名、陣型與移動方向箭頭、即時戰況快報。
- **自由運鏡**：滑鼠左鍵旋轉、滾輪縮放、右鍵平移；可切換「自動運鏡」電影運鏡或「自由視角」。
- **兩種模式**：歷史節目自動播放 / 玩家互動探索。

## 操作

| 操作 | 說明 |
|------|------|
| 左鍵拖曳 | 旋轉鏡頭 |
| 滾輪 | 縮放 |
| 右鍵拖曳 | 平移 |
| 點擊船艦 | 查看單位資訊 |
| 時間軸節點 | 跳至該幕 |
| ▶ / ⏸ | 自動播放 / 暫停 |
| ⏮ ⏭ | 上一幕 / 下一幕 |
| 自動運鏡 / 地名標示 | 切換顯示 |

## 技術

- [Three.js](https://threejs.org/) r160（CDN，ES module + importmap）
- 純單檔 `index.html`，無建置流程；以靜態網頁部署於 GitHub Pages。
- 字型：Noto Sans TC / Noto Serif TC。

> 註：赤壁之戰發生於三國時代（西元 208 年），尚無火藥兵器。本作為戲劇化呈現，將「火攻」延伸表現為火箭、火砲齊發等特效，僅供觀賞，非史實考據。
